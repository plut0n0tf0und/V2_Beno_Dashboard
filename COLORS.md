# Project Accent Colors

## Accent (both themes)
- **Primary accent / interactive**: `#679CFF` (blue) — buttons, active states, rings, chart fills, tertiary token
- **Error**: `#EC7C8A` (soft red)

## Light Mode
- Surface base: `#FFFFFF` → containers up to `#DADCE0`
- Text: `#1F1F1F` (on-surface), `#5F6368` (variant)
- Primary label color: `#C6C6C7`

## Dark Mode
- Surface base: `#0E0E0E` → containers up to `#252626`
- Text: `#C6C6C7` (on-surface), `#ACABAA` (variant)
- Primary label color: `#C6C6C7`

## Key token → hex (dark)
| Token | Hex |
|---|---|
| `tertiary` | `#679CFF` |
| `on-surface` | `#C6C6C7` |
| `on-surface-variant` | `#ACABAA` |
| `surface` | `#0E0E0E` |
| `surface-container` | `#191A1A` |
| `error` | `#EC7C8A` |
